# Smart Disease Prediction System

## Overview
This project predicts diseases based on user symptoms using machine learning.

## Features
- Disease prediction
- User-friendly GUI
- Symptom analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Tkinter

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
